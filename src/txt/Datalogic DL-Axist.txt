ANDROID Factory Reset                                                                    ANDROID
------------------------------------------------------------------------------------------------
 Using the System Settings:
 -> Settings -> Backup & Reset -> Factory Reset

 Usinge the Recovery Menu:
 -> Access the Recovery Menu and select Factory Reset


------------------------------------------------------------------------------------------------
ANDROID Firmware Upgrade                                                                 ANDROID
------------------------------------------------------------------------------------------------
 With DXU (Datalogic Configuration Utility for Android devices):
  Connect the device to DXU and perform the Upgrade
  
 With System Upgrade / Local Upgrade in System Settings:
  Copy the Upgrade File to the device storage
  Select Settings -> System Upgrade -> Local Upgrade
  Select the upgrade.zip file
  Perform a Upgrade


------------------------------------------------------------------------------------------------
ANDROID Accessing the Recovery Menu                                                      ANDROID
------------------------------------------------------------------------------------------------
 During BOOT press and hold the FUNCTION BUTTON
 
 In Recovery mode you can do the following:
 > Reboot system now
 > Apply update from external storage
 > Factory Reset
 > Enterprise Reset
 > OTA Update Options
 > Secure Settings
