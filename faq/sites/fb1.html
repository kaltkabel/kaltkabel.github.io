2. ADB Method

  • 1. adb devices
  • 2. adb reboot recovery
  • 3. auf dem Gerät "apply update from adb" auswählen
  • 4. adb sideload "xxx.zip"
  • 5. "reboot system now" auswählen

3. GMS - non GMS

GMS steht für Google Mobile Services. Diese sind abhängig von den Geräten, ob die mit installiert werden müssen oder nicht.

4. Windows CE / Embedded Handheld (Factory Reset)

Ein Factory Reset kann hier durch die Clean Files ausgeführt werden, die es auf dem Fileserver gibt (T:\Technik\_Fachbereich-1). Diese müssen dann auf dem Gerät in den "Temp" Ordner kopiert werden und anschließend vom Gerät aus, ausgeführt.

Alternative: man führt auf dem Gerät eine Tastenkombi aus, die das Gerät auf FC setzt.

Auf dem Fileserver gibt es eine HTML-Seite (T:\Technik\_DEMODIENST\Mobile Terminals\JT Demo Tool WIP\index.html), wo zu fast allen Geräten beschrieben wird, wie man ein Factory Reset durchführt oder T:\Lager\Rüstabteilung\Spars,Software\Datalogic

Zudem gibt es dort noch weitere Verzeichnisse mit hilfreichen Informationen und allen Versionen
