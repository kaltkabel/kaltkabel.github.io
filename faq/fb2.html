Druckersprachen

Viele Hersteller können andere Druckersprachen emulieren, jedoch sollte man, soweit möglich, die Herstellerspezifische Sprache des Druckers verwenden um Fehler auszuschließen.

Bei Ansteuerung der Etikettendrucker durch Windowsdruckertreiber übersetzen diese das zu druckende Layout in die für den Drucker verständliche Sprache. Da die meisten Programme die Daten als Grafik ausgeben (ausnahmen sind die Druckertreiberhersteller eigenen Layoutprogramme, z.B. Bartender von Seagull, hier werden die Layoutobjekte, soweit möglich, in Objekte der Druckersprache umgesetzt), ist es sinnvoll, bei Barcodes diese direkt mittels der Druckersprache zu generieren, um Pixelfehler durch Falschberechnung und/oder Rundungsfehler auszuschließen.

Um die Druckfiles zu erzeugen können in vielen Programmen Skripte oder Dateien hinterlegt werden, welche die Erzeugung der Druckfiles übernehmen. Diese werden dann direkt an den Drucker gesendet.

  • Zebra verwendet ZPL und teilweise noch EPL
  • Datamax benutzt DPL
  • TSC benutzt TSPL
  • Intermec benutzt IPL oder Fingerprint

Evolis Ersatzteile

Wenn Evolis Ersatzteile bestellt werden, werden die ausgetauschten Ersatzteile mit einer RMA bei Evolis beantrag, damit wir von Evolis eine Gutschrift erhalten.

Evolis Commands

Pip = Wert setzen
Rip = Wert lesen.

  • Rmp;all = Factory Defaults
  • Pip;DHCP = Set the printer in DHCP mode
  • Pip;192.168.1.1 = Set the IP address
  • Penm;255.255.255.0 = Set the network mask
  • Pegw;192.168.1.254 = Set the default gateway
  • Pmip;224.69.86.79 = Set the multicast IP
  • Snetw = Validate settings
