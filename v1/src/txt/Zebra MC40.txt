 COLDBOOT AND
---------------
1. Simultaneously press POWER + LEFT SCAN / ACTION + VOLUME UP
2. The Device will reboot



 FACTORY RESET AND
-------------------
1. Copy files
	 JB: 40N0JxxFRxxxxxxx.zip -> microSD:\
	 KK: 40N0KxxFRxxxxxxx.zip -> microSD:\
	 or
	 Use ADB directory
2. Boot into RECOVERY
	 Press and hold LEFT SCAN + LEFT ACTION button while booting
	 or
	 ADB command: "adb reboot recovery"
3. Apply UPDATE
	 Select "apply update from SD"
	 Select the xx.zip file
	 or
	 Select "apply from ADB"
	 ADB command: "adb sideload xx.zip"
